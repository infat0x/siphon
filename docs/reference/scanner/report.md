# Report Generator (`scanner/report.go`)

The `report.go` module aggregates, deduplicates, and formats the raw findings from all scanner modules into human-readable text and structured JSON formats.

## Overview

A scanner is only as good as its output. `report.go` ensures that Siphon's results are presented clearly, sorted by severity, and stripped of duplicates. 

> [!NOTE]
> The module generates two files simultaneously: a stylized `.txt` report for human review, and a structured `.json` report for integration into CI/CD pipelines or SIEMs.

## Deduplication and Filtering

Before writing the report, Siphon performs a final pass over the findings. 
It filters out any matches shorter than 12 characters and double-checks them against the `FalsePositiveRe`. It then calls `core.DedupFindings()` to remove identical findings discovered across different files or by different scanning modules.

## Sorting Hierarchy

Findings are sorted primarily by **Severity**, and secondarily by **Confidence**.

```go
severityOrder := map[string]int{"CRITICAL": 0, "HIGH": 1, "MEDIUM": 2, "LOW": 3, "INFO": 4, "": 5}
sort.Slice(highConf, func(i, j int) bool {
    si := severityOrder[highConf[i].Severity]
    sj := severityOrder[highConf[j].Severity]
    if si != sj {
        return si < sj
    }
    return highConf[i].Confidence > highConf[j].Confidence
})
```

## Output Formatting

The text report includes:
1. **Execution Stats**: Live hosts, URLs collected, JS files downloaded, and download success rate.
2. **Severity Breakdown**: A quick summary of how many Critical/High/Medium issues were found.
3. **Scanner Breakdown**: How many findings were generated by `regex`, `entropy`, `inline-scanner`, etc.
4. **Detailed Findings**: Grouped by Severity, then by finding Type (e.g., "AWS Access Key"), displaying the exact file, line number, entropy, confidence, and the matched string (truncated to 150 characters).

> [!TIP]
> The JSON report contains all the exact same data, structured cleanly under a `"findings"` array, making it perfect for automated parsing tools like `jq`.
